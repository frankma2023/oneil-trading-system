## 📈 抛盘日规则速查表

| 类型 | 条件 | 计数 |
|------|------|------|
| **平盘日** | \|涨跌幅\| < 0.05% | 0 |
| **标准抛盘日** | 跌幅 ≤ -0.1% + 成交量↑ | 1 |
| **重抛盘日** | 跌幅 ≥ -1.5% + 成交量↑ | 2 |
| **特殊抛盘日** | 涨幅 < 0.2% + 最高≥0.5% + 成交量↑1.3倍 + 上影线够长 | 1 |
| **反转抛盘日** | 收阴 + 最高≥0.5% + 成交量↑1.2倍 + 上影线够长 | 1 |
| **确认日** | 涨幅 ≥ 1.5% + 成交量↑ | 抵消1个 |
