## 📊 抛盘日规则要点
1. 平盘日过滤: |涨跌幅|<0.05% → 不计算
2. 标准抛盘日: 跌幅≤-0.1% + 放量 → 计1个
3. 重抛盘日: 跌幅≥-1.5% + 放量 → 计2个
4. 确认日抵消: 涨幅≥1.5% + 放量 → 抵消1个
5. 25日窗口: 严格滚动25交易日
