# 欧奈尔投资系统 - 目录结构说明

## 📁 七层架构概览

```
investment/                    # 项目根目录
├── data/                     # 数据层 - 所有数据相关
├── core/                     # 核心层 - 欧奈尔算法和回测
├── api/                      # API层 - FastAPI Web服务
├── web/                      # 前端层 - 7个功能页面
├── config/                   # 配置层 - 所有参数配置
├── scripts/                  # 脚本层 - 自动化任务
├── tests/                    # 测试层 - 完整测试套件
├── docs/                     # 文档层 - 详细文档
└── logs/                     # 日志层 - 系统日志
```

## 📊 各层详细说明

### 1. 数据层 (`data/`)
**职责**: 统一数据访问，数据库管理，API接口封装

```
data/
├── database/                 # 数据库文件
│   └── lixinger.db          # SQLite数据库 (10年K线数据，从旧系统复用)
├── api/                      # API客户端
│   └── lixinger/            # 理杏仁API脚本 (从旧系统精选复用)
│       ├── base_api.py      # API基础类
│       ├── api_stock_candlestick.py    # 个股K线接口
│       ├── api_index_candlestick.py    # 指数K线接口
│       └── db_manager.py    # 数据库管理
└── access.py                # 统一数据访问接口 (全新开发)
```

**关键文件**:
- `access.py`: 提供统一的数据访问API，封装所有数据库操作
- 数据库: `lixinger.db` (2.7GB，包含10年历史数据)
- API脚本: 仅复用核心数据获取功能，不依赖业务逻辑

### 2. 核心层 (`core/`)
**职责**: 欧奈尔投资算法实现，回测框架，业务逻辑

```
core/
├── market/                  # 大盘扫描引擎 ✅ 完成
│   ├── __init__.py
│   └── indicators.py       # 抛盘日/吸筹日/追盘日识别
├── industry/                # 行业扫描引擎 (待开发)
│   └── __init__.py
├── stocks/                  # 个股筛选引擎 (待开发)
│   └── __init__.py
├── patterns/                # 形态识别引擎 (待开发)
│   └── __init__.py
├── backtest/               # 回测框架 ✅ 完成
│   ├── __init__.py
│   └── framework.py        # 科学回测引擎
├── portfolio/               # 投资组合管理 (待开发)
│   └── __init__.py
└── utils/                   # 工具函数 (待开发)
    └── __init__.py
```

**已完成**:
- `market/indicators.py`: 完整的欧奈尔市场指标计算
  - 三种抛盘日识别 (标准、假阳线、日内反转)
  - 吸筹日识别
  - 追盘日识别
  - 市场健康度评分 (0-100)
  - 25日滑动窗口计数
- `backtest/framework.py`: 科学回测框架
  - 参数化策略回测
  - 多维度绩效评估 (夏普比率、最大回撤等)
  - 参数扫描优化
  - 结果缓存和比较

### 3. API层 (`api/`)
**职责**: Web服务接口，RESTful API

```
api/
└── main.py                 # FastAPI主应用 ✅ 完成
```

**已完成**:
- `main.py`: FastAPI应用，提供以下API端点:
  - `GET /api/health`: 健康检查
  - `GET /api/market/dates`: 交易日列表
  - `GET /api/market/analysis`: 市场分析
  - `GET /api/market/indices`: 指数列表
  - `GET /api/stocks/search`: 股票搜索
  - `GET /api/stocks/{code}`: 股票数据
  - `GET /api/backtest/strategies`: 回测策略列表
- 自动API文档: http://127.0.0.1:8000/docs
